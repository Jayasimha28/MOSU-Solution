# MySQL StatefulSet on Local Kubernetes

## Coming Soon

MySQL StatefulSet deployment for local Kubernetes clusters (Minikube, Kind, K3s, MicroK8s, Docker Desktop).

### Local Storage Options
- **hostPath**: Simple local directory mounting
- **local-path-provisioner**: Dynamic local storage provisioning
- **OpenEBS**: Cloud-native storage for Kubernetes
- **Longhorn**: Distributed block storage
- **Rook-Ceph**: Storage orchestration

### Supported Local K8s Platforms
- Minikube
- Kind (Kubernetes in Docker)
- K3s (Lightweight Kubernetes)
- MicroK8s
- Docker Desktop Kubernetes
