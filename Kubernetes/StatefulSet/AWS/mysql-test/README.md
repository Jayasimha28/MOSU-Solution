# MySQL StatefulSet on AWS EKS

## Coming Soon

MySQL StatefulSet deployment configurations for AWS EKS with EBS storage classes.

### AWS Storage Classes
- **gp2**: General Purpose SSD (default)
- **gp3**: General Purpose SSD (better performance/cost)
- **io1/io2**: Provisioned IOPS SSD
- **st1**: Throughput Optimized HDD
- **sc1**: Cold HDD
